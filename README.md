# Weekend-Getaway-Ranker

A Python script to recommend top weekend destinations in India from a given city.

**Features**

Enter a source city.

Ranks cities based on rating, popularity, and proximity.

Shows results in a table with city details and scores.

**How to Run**

**Install dependencies:**

pip install -r requirements.txt


**Run the script:**

python weekend_ranker.py


Enter your source city when prompted.
